2.1 Правила P (окончательная версия)
		
1	‹START› → const ‹CONST›	ключевое слово
2	‹CONST› → ␠ ‹SPACE›	ровно один пробел после const*
3	‹SPACE› → letter ‹ID›	идентификатор начинается с буквы
4	‹ID› → letter ‹ID› | ‹TYPE›	буквы внутри ID или сразу переходим к типу
5	‹TYPE› → f32 | f64 ‹EQUAL›	поддерживаем два типа
6	‹EQUAL› → = ‹INT›	символ =
7	‹INT› → digit ‹INTREM›	≥ 1 цифра перед точкой
8	‹INTREM› → digit ‹INTREM› | . ‹FRAC›	целая часть может быть длинной
9	‹FRAC› → digit ‹FRAC› | ;	≥ 1 цифра после точки, затем ;


2.2 Дополнительные обозначения (без изменений)
•	‹Digit› → “0” | “1” | … | “9”
•	‹Letter› → “a” | … | “z” | “A” | … | “Z”
2.3 Компоненты грамматики (обновлено)
•	Z (стартовый символ) = ‹START›
•	VT = { const, f32, f64, =, ;, ., 0–9, a–z, A–Z, ␠ }
•	VN = { ‹START›, ‹CONST›, ‹SPACE›, ‹ID›, ‹TYPE›, ‹EQUAL›, ‹INT›, ‹INTREM›, ‹FRAC› }
